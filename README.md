# PixelVault

A personal photo gallery with ImageKit storage, auto-thumbnails, and a secret upload page.

## Features

- Masonry grid gallery with smooth fade-in
- Click any photo to open full-res lightbox
- Prev/next navigation + keyboard arrows
- One-tap download
- Secret upload page — works on mobile and desktop
- Thumbnails auto-generated by ImageKit (no extra storage used)
- Demo mode when ImageKit is not configured

## Tech Stack

- Next.js 15 (App Router)
- ImageKit (storage + CDN + auto thumbnails)
- Tailwind CSS
- Vercel (hosting)

## Setup

### 1. ImageKit

1. Sign up free at [imagekit.io](https://imagekit.io) — no credit card needed
2. Go to **Developer Options** → copy:
   - URL Endpoint (`https://ik.imagekit.io/your_id`)
   - Public Key
   - Private Key

### 2. Environment variables

Copy `.env.local.example` to `.env.local` and fill in:

```
NEXT_PUBLIC_IMAGEKIT_URL_ENDPOINT=https://ik.imagekit.io/your_id
NEXT_PUBLIC_IMAGEKIT_PUBLIC_KEY=public_xxxx
IMAGEKIT_PRIVATE_KEY=private_xxxx
UPLOAD_SECRET=your-secret-upload-path
```

> ⚠️ `.env.local` is gitignored — your keys are never pushed to GitHub.

### 3. Install & run

```bash
npm install
npm run dev
```

### 4. Deploy to Vercel

Push to GitHub, import on [vercel.com](https://vercel.com), and add the 4 env variables in:
**Project → Settings → Environment Variables**

## Usage

| Page | URL |
|---|---|
| Gallery (public) | `yoursite.com/` |
| Upload (only you) | `yoursite.com/upload/your-secret` |

The upload page works on mobile — tap the box to open your camera roll or camera directly. Select multiple photos and upload in one go.
